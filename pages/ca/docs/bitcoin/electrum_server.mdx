## Bitcoin amb dispositius de signatura

La millor manera de mantenir de manera segura els vostres parells de claus de
bitcoins (és a dir, la millor combinació de seguretat i usabilitat) és utilitzar
un dispositiu de signatura (com [SeedSigner](https://seedsigner.com),
[Passaport](https://foundationdevices.com),
[BitBox](https://shiftcrypto.ch/bitbox02), [Coldcard](https://coldcard.com),
[Jade](https://blockstream.com/jade/) o [Trezor](https://trezor.io)) en
ombinació amb el vostre propi node Bitcoin. Això us proporciona seguretat,
privadesa i elimina la necessitat de confiar en un tercer per verificar les
transaccions.

El client de Bitcoin al mateix Microbolt no està destinat a contenir fons.

Una possibilitat d'utilitzar el client Bitcoin amb els vostres dispositius de
signatura de Bitcoin és utilitzar un servidor Electrum com a programari
intermediari. Importa dades del client Bitcoin i les proporciona al programari
de signatura que admet el protocol Electrum. Aplicacions de signatura
d'escriptori com [Sparrow](https://sparrowwallet.com),
[BitBoxApp](https://shiftcrypto.ch/app), [Electrum](https://electrum.org) o
[Specter Desktop](https://specter.solutions/desktop) que admet els dispositius
de signatura es pot utilitzar amb el vostre propi node sobirà de Bitcoin.

Trieu el vostre servidor, quan hàgiu acabat, aneu a la secció de Cartera
d'escriptori
